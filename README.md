# 🌐 Network Tic-Tac-Toe  
**Multiplayer game via TCP sockets in C**  

## Features  
- 🖥️ **Server-client architecture**
- 🎮 **Modes**: Player vs Player
## How to Run  
**Build server and client and run it**:  
   ```bash
   gcc server.c -o server && gcc client.c -o client
   ./server
   ./client
   ```
